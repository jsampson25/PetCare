# Cross-Application Tests

End-to-end, contract, security, and cross-domain tests will live here when they cannot be owned cleanly by a single application or package.

Priority journeys include:

1. Business setup to first available service
2. Customer registration and pet creation
3. Booking with eligibility checks and deposit
4. Waitlist promotion
5. Check-in through checkout
6. Payment webhook reconciliation
7. Tenant-isolation verification
